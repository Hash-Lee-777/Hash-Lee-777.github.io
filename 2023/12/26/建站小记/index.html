四、部署Github Page

Github Page是提供静态网站访问服务的，就是将静态网站项目传到GitHub上，它能为你提供一个url入口（如Hash-Lee-777.github.io），直接访问你的静态网站，而Hexo编译生成的就是一个静态网站。
